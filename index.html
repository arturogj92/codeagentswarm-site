CodeAgentSwarm.

A supervised desktop workspace for running multiple AI coding CLI agents in parallel, on macOS and Windows.

Visit codeagentswarm.com →

What it is

CodeAgentSwarm lets you run several AI coding agents side by side in parallel terminals, instead of one at a time, while keeping full visibility and control over what each one is doing. It runs on top of your existing AI coding CLIs, it does not replace them.

Core features

  • Run multiple AI coding terminals in parallel, each with its own agent.
  • Desktop notifications the moment an agent finishes a task or needs your input.
  • Searchable conversation history across every agent and project.
  • Per-terminal live file diffs so you can review changes before they land.
  • A task/kanban board the agents update over MCP.
  • Permission and Turbo mode controls to keep routine work fast and sensitive operations safe.

Availability

Available for macOS (Apple Silicon) and Windows (x64 and ARM64). CodeAgentSwarm is currently free while in open beta. You can download it and read the full guides at codeagentswarm.com.
